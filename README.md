# 🧪 語詞實驗室 (Word Lab)

這是一個專為國小學生設計的互動式語詞學習網頁遊戲。透過「字形合成」、「詞語提煉」與「配方鑑定」三個關卡，幫助學生在有趣的鍊金術情境中掌握易混淆字與詞語用法。

-----

## 🚀 立即開始遊戲
 [👉 點擊此處開始遊戲 👈](https://weddy1225-blip.github.io/AI-Chinese04/)

-----

## 🌟 遊戲特色
- **字形合成**：互動式部首與材料組合，強化對漢字結構的記憶。
- **詞語提煉**：情境化的選字填空訓練。
- **實驗報告**：自動記錄學習過程中的錯誤題目，方便老師或家長後續指導。
- **響應式設計**：自動置中佈局，適合教室電子白板或電腦操作。

## 🚀 如何執行
只需下載 `index.html` 並使用任何瀏覽器開啟即可開始實驗！

- 原生 JavaScript (Vanilla JS)
